# BERT Model

The BERT model was trained using `bert-base-uncased` from the Hugging Face Transformers library.

The full saved BERT model file is not included in this repository because the `model.safetensors` file is larger than GitHub's normal file size limit.

Included BERT-related files:

- `notebooks/Bert_Model.ipynb`
- `results/bert_evaluation_summary.csv`
- `results/fresh_absa_results.csv`
- `results/rotten_absa_results.csv`

To reproduce the BERT model, run the `Bert_Model.ipynb` notebook.
