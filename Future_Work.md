## Future Work

Future improvements can include training and evaluating the BERT model on a larger dataset or the full dataset to improve reliability. Other transformer models such as RoBERTa, DistilBERT, ALBERT, and DeBERTa can also be compared to identify the best model for movie review sentiment classification.

The BiLSTM model can be improved using pretrained word embeddings such as GloVe, FastText, or Word2Vec. Further tuning of hyperparameters such as LSTM units, dropout rate, batch size, learning rate, and sequence length may also improve performance.

Additional machine learning models such as Naive Bayes, Random Forest, XGBoost, and ensemble classifiers can be tested. The aspect-opinion extraction process can also be improved using a more advanced ABSA model instead of simple POS-tag patterns.

Finally, the best-performing model can be deployed into a simple web application or dashboard where users can enter a movie review and receive an automatic Fresh or Rotten prediction.
